# Trabajo Práctico N° 2 - Curso Nivel 2

## Enunciado

Crear una aplicación que utilice el componente web y se conecte a la siguiente URL mediante un GET.


  Esto devolverá un JSON con el siguiente formato:
  
	{ persona: {
		“nombre”: “Juan”,
		“apellido”: “Perez”,
		“edad”: 20
	 } }
   
La aplicación deberá mostrar los datos de la persona en pantalla.
